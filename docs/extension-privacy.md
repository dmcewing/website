# Privacy Policy

## Privacy Policy (for all extensions)
1. This extension does not collect any user data
2. This extension does not sync any data to any remote server. It only uses local storage (chrome.storage.local or window.localStorage)
3. This extension only redirects to pages already available in the current Dynamics 365 for Operations session, for which the user must have already logged in before activating the extension.
4. This extension does not monitor the user.
5. The current URL of the active tab is interrogated to determine if the extension should be enabled or disabled.  This information is not stored other than required for the functionality described in points 2 and 3 above.
